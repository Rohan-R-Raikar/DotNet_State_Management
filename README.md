# State Management Projects

This repository contains .NET projects focused on **state management** in web and console applications.

## Tech Stack
- .NET 6 / .NET 7
- C# / ASP.NET
- Real-time updates and state management in shopping cart and multi-database scenarios

## Projects Included
- ShopingCartStateManagement — Basic state management demo
- LiveShoppingCart‑RealTime — Real-time shopping cart application
- OneDBMultipleURL — Managing multiple database URLs dynamically
- SampleNoSolution — Sample project demonstrating state handling concepts

## Notes
- Secrets (API keys, tokens, DB passwords) should **never be committed**.
- Projects are primarily learning/demo apps for practicing state management patterns.
